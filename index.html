What It Does

🌍

Multi-Engine Translation

Powered by DeepL, MyMemory, and AI — switch between providers with one click.

🔤

Smart Conjugation

Type "yo tener" or "jorge venir" and get the correct conjugated verb in your target language via AI.

📚

Learned Phrases

Save translations you use often. They're cached locally for instant lookup — no API call needed.

⚡

Lightning Fast

In-memory caching, batched storage writes, and debounced UI — every interaction is optimized for speed.

🌙

Dark Mode

Easy on the eyes at night. Toggle with one click — settings persist across sessions.

📋

Inline or Panel

Replace text directly on the page, or use the draggable panel for side-by-side viewing.

Get the Free Beta

This is an early preview. All features are free during beta.

1

Download the Extension

Grab the latest beta from the GitHub Releases page and extract it somewhere on your computer.

2

Open Chrome Extensions

Type chrome://extensions/ into your address bar and press Enter.

3

Enable Developer Mode

Toggle the switch in the top-right corner of the page.

4

Load Unpacked

Click "Load unpacked" and select the folder where you extracted the extension.

5

Start Translating!

Select any text, right-click → Translate, or press Alt+T.
